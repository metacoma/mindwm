How-to (WIP)

---------------------------------------------------

```bash
$ cd ~
$ git clone https://github.com/metacoma/mindwm
$ cd mindwm
$ ./kapitan compile --fetch
$ . compiled/logstash/functions.bash
$ logstash_build
$ tmux_session_start 
$ <Ctrl-b + d> #detach from tmux session
$ . compiled/fluentd/function.bash
$ tmux_session_start
```
